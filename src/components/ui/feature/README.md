# UI Feature Components

This directory contains feature-specific components that are tailored to particular use cases in the application.

## Components

- **CandidateTableSkeleton** - Loading state for candidate table
- **EmptyState** - Illustrated empty state with actions
- **UploadProgressList** - Multi-file upload progress display
- **FilterPresetManager** - Save and load filter presets

## Usage

Feature components should:
- Be specific to a particular feature or page
- Combine primitives and composites
- Handle feature-specific logic
- May not be reusable across different features
